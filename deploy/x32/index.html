Deploy a shared x32 iPad

  1. Supervise & disallow pairing using Apple Configurator.
  2. Set up the device as normal, signing in with Apple ID.
  3. Install GuitarTuna, Tuner Lite, QLab Remote, Mixing Station, X32-Mix
  4. Uninstall all apps possible to remove apart from Clock, Settings, Safari, Notes, Photos, Camera.
  5. All remaining apps go in a "Bloat" folder. This goes first on the home screen, followed by Clock, Settings, Safari, Notes, GuitarTuna, Tuner Lite, QLab Remote
  6. In the dock, place Photos, Camera, Mixing Station, X32-Mix in order.
  7. Disable iCloud syncing features, apart from Find My. Finally, install the profile.
